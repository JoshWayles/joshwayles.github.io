Teddy Swims gift box

A commission for the Co-op Live arena in Manchester to create a set of tattoo inspired designs to be applied to a gift box for the musician Teddy Swims. Teddy is an avid tattoo collector so they wanted to gift him something to reflect that passion. The box was filled with curated books by the Manchester based, visual arts bookshop, Unitom.
